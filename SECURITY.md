# Security Policy

## Supported Versions

This repository is currently in pre-release development.
Security policy will follow the same model as the upstream [ymnao/scripta](https://github.com/ymnao/scripta) once released.

| Version | Supported |
|---------|-----------|
| 0.x     | Pre-release / development |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through GitHub Security Advisories on the upstream repository:
https://github.com/ymnao/scripta/security/advisories/new

**Do not open a public issue for security vulnerabilities.**

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
